# Deckbuilding Game

A deckbuilding game prototype built with React, Vite, Tailwind CSS, and boardgame.io.

## Setup

Install dependencies:
```bash
npm install
```

Run the development server:
```bash
npm run dev
```

## Features

- Draw cards from your deck to your hand
- Play cards from your hand to your discard pile
- Beautiful UI with Tailwind CSS
- Game state management with boardgame.io
